# Github学生优惠获取教程

步骤(1-4 推荐使用电脑操作)

## 填写信息(可以使用VPN)

1. 点击头像，选择`settings`
   1. 可选菜单：Set status、Profile、Repositories、Stars、Gists、Organizations、Enterprises、Sponsors
   2. 进入Settings后可见：Copilot settings、Feature preview（New）、Appearance、Accessibility、Try Enterprise Free、Sign out、Public profile
2. 填写公开资料信息
   1. 名字填写要求：姓在后名在前，如:HuaLi
   2. 需填写学校官网网址

![image-20251212214512055](https://cdn.jsdelivr.net/gh/Wanglihan954/Picture-bed@img/img/image-20251212214512055.png)

## 完善账单信息(可以使用VPN)

1. 进入账单信息页面：在设置中找到`Billing information`（账单信息），入口路径包含Public profile（公开资料）、Payment information（支付信息）、Account（账户）、Appearance（外观）等多个设置分类
2. 填写账单信息项
   1. 姓名：First name（名）、Last name（姓），示例为Hua Li
   2. 地址（Street, P.O.box）：填写大学英文名，可重复填写，示例为The University of Tokyo
   3. Address line 2（Apartment, suite, unit）：无示例，按需填写
   4. City（城市）：填写英文，示例为Tokyo
   5. Country/Region（国家/地区）：可选项如Japan，国内用户选China
   6. State/Province（州/省）：按需填写
   7. Postal/Zip code（邮政编码）：部分国家/地区必填（美国需9位邮编）
   8. VAT/GSTID（增值税/商品及服务税识别号）：按需填写
3. 填写完成后点击`Save billing information`（保存账单信息）

![image-20251212214544933](https://cdn.jsdelivr.net/gh/Wanglihan954/Picture-bed@img/img/image-20251212214544933.png)

## 材料准备

需准备**Student Verification Report**（学生验证报告），内容需包含以下信息，填写个人实际内容即可：

```Plain
Student Verification Report
Name : Han Ning
School : Peking University
Student Number : 123456789
Graduation Date :June 20, 2028
Study Form: Full-time
```

（注：文档中出现两份仅学号有差异的报告模板，实际准备一份个人真实报告即可）

## 学生邮箱添加(可以使用VPN)

1. 进入邮箱设置板块，若已添加学校邮箱但未显示通过，需进行验证
2. 操作入口：找到`Add email address`（添加邮箱地址）选项，录入学校邮箱并完成学校邮箱验证
   1. 示例邮箱格式：[20230001@pku.edu.cn](mailto:20230001@pku.edu.cn)（仅为格式参考，需替换为真实学校邮箱）

![image-20251212214904510](https://cdn.jsdelivr.net/gh/Wanglihan954/Picture-bed@img/img/image-20251212214904510.png)

## 双因素认证(可以使用VPN)

1. 进入双因素认证设置入口：在设置中找到`Password and authentication`（密码和认证）分类下的`Two-factor authentication`（双因素认证）

2. 认证流程

   1. 进入后会显示二维码，需在手机上下载**Authenticator**类身份验证App（如Microsoft Authenticator）

   ![image-20251212215007038](https://cdn.jsdelivr.net/gh/Wanglihan954/Picture-bed@img/img/image-20251212215007038.png)

   1. 打开App扫码完成验证即可启用双因素认证，增强账户安全性

![image-20251212214934108](https://cdn.jsdelivr.net/gh/Wanglihan954/Picture-bed@img/img/image-20251212214934108.png)



## 开始申请(开始使用手机进行申请)

注意：此步骤全程**不能使用VPN**，且需关闭校园网WiFi，仅使用国内流量；因GitHub IP限制，可能出现登录不畅，可多尝试几次

1. 进入教育福利申请页面，选择教育身份：`Student`（学生，勿选Teacher）
2. 若已完成学校邮箱验证，页面会显示验证状态，确认学校选择后点击`Continue`
3. 选择证明类型（==需选择b==），可选类型如下：
   1. 注明日期的学校证件-完整
   2. Dated official/unofficial transcript -Fair
   3. Dated enrollment letter on school letterhead -Fair
   4. Dated class schedule for the semester-Poor
   5. Dated syllabus for a class - Poor
   6. Dated receipt from bursar（可在此选项下上传证明材料照片，仅需拍摄文字材料部分）
4. **材料上传/提交完成即可，期间网站可能出现访问不畅，属于正常现象，可多次尝试**
